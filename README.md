# CompTIA A+ Core 1 (220-1201) Memorization Reference

This repository hosts a single-page, CSS-styled HTML reference sheet designed for rapid study and memorization of the **CompTIA A+ Core 1 (220-1201)** exam objectives.

## 📋 What's Included
This reference covers "must-know" facts that frequently appear as "traps" or "triggers" on the exam:
- **Domain 2 (Networking):** TCP/UDP Port numbers, Wi-Fi standards (802.11a/b/g/n/ac/ax), and IPv4/APIPA ranges.
- **Domain 3 (Hardware):** Cable categories (Cat 5e–6a), T568A/B wiring diagrams, RAM generations, and RAID configurations.
- **Domain 5 (Troubleshooting):** Symptom-to-cause quick reference for hardware, network, and display issues.
- **Printer Tech:** A detailed breakdown of the 7-step Laser Printing (EP) process.

## 🚀 How to Use
1. **Live View:** https://gmtrends.github.io/CompTIA_A-_CORE1_Reference
2. **Local View:** Download the `index.html` file and open it in any modern web browser.
